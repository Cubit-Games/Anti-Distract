Welcome to the GitHub page for Block-Distract

One day I was sitting down doing my online school, and hey, i thought about Youtube...next thing you know iv been on youtube for 2 hours and didn't get school done. That is one of the reasons for this program! This program blocks either a preset of distracting sites or allows you to block your own custom site.

How Does It Work?

Inside of Windows there is a file at "C:\Windows\System32\dirvers\etc\hosts". This "hosts" file allows you to redirect any URL to an IP Address. What this program does it take the site you want to block and then has the "hosts" file redirect to a non-existent IP{0.0.0.0).

What Are The Presets?

Right now the presets are

Youtube

Search Engines(Google, Bing, Yahoo, Ask)

Social Media(Facebook, Instagram, Twitter, GooglePlus, Tumblr, Pinterest, and Vine)

Email(Gmail, Yahoo, Outlook(Hotmail), Mail.com)

All(All sites listed here, not the whole web)

There is as also a "Block Custom Site" option which allows you to block your own site that you put the URL in for. Of course there is also an option to make every site available again.
